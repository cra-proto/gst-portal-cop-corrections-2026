Get a GST/HST access code – File your GST/HST return

You may be looking for:

  • NETFILE access code for personal income tax

The GST/HST access code is a 4-digit code required for some GST/HST filing methods. It is a personal security key that authenticates you when you are not signed into a CRA account. You should not give your access code to anyone unauthorized.

Your GST/HST access code stays the same for every filing period unless you change it.

On this page

  • Filing without an access code
  • When you need an access code
  • How to get or change your access code

Filing without an access code

You do not need an access code if you are filing:

  • In your CRA account (My Business Account or Represent a Client)
  • Through a participating Canadian financial institution
  • By mail

When you need an access code

You do need an access code only if you are filing:

  • Online using the GST/HST NETFILE form (outside of a CRA account)
  • Using third-party accounting software (GST/HST Internet File Transfer)
  • Over the phone using GST/HST TELEFILE

How to get or change your access code

You may have received your access code by mail:

  • If you ever received a GST/HST filing package by mail
  • When you registered for the GST/HST, if you are a non-resident

If you don't have your access code, or need a new one, there are 3 ways to get or change it:

  • Get your access code in your CRA account (business account only)
  • Change your access code using the online form, as long as you have filed a GST/HST return before
  • If you cannot use the other options, get or change your access code by phone
  CRA Account
  1. Sign in to your CRA account
  2. Select your Business account
  3. Select GST/HST and your RT number
  4. On the GST/HST account overview page, select the View return details button

Representatives who are accessing your account though their representative account will not see your access code, regardless of their authorization level.

  Online form

If you have filed a GST/HST return before, change your access code using the online form:

  1. Visit GST/HST access code online
  2. Enter your business number
  3. Enter information from a return you filed for a reporting period ending within the last 3 years, either:
    • The reporting period and line 109 (net tax), if the amount on line 109 is not $0.00
    • The reporting period and confirmation number, if you filed the return electronically
  4. Enter the new access code you want
  5. Use your new access code to file all future GST/HST returns
  By phone

If you cannot sign in to your CRA account or use the online form, call us to get or change your access code:

  • In Canada or the United States: 1-800-959-5525
  • In Yukon, Northwest Territories or Nunavut only: 1-866-841-1876
  • Outside Canada or the United States, call us collect: 613-940-8497
